# Дурак Онлайн — 3 визуальных направления

Документ фиксирует три альтернативные визуальные линии для Telegram Mini App.  
Все варианты опираются на единую UX-структуру текущего проекта и отличаются стилем, цветом, типографикой и композицией.

## Direction A — Neon Felt

Идея: цифровой карточный зал с акцентом на контраст и стеклянные слои.

- Палитра:
  - `bg`: `#010A1B`
  - `surface`: `rgba(128,138,255,0.20)`
  - `accent`: `#808AFF`
  - `success`: `#12EB53`
  - `danger`: `#FF5555`
- Типографика:
  - `display`: SF Pro Display Bold
  - `body`: SF Pro Display Regular
- Визуальные маркеры:
  - мягкие glow-границы карточек
  - акцентные pill-кнопки с градиентом
  - темный войлочный стол на игровом экране
- Лучшее применение:
  - основной публичный стиль для live-версии

## Direction B — Light Casino Editorial

Идея: светлый, более «премиальный» интерфейс с чистой сеткой и мягкими тенями.

- Палитра:
  - `bg`: `#F5F6FC`
  - `surface`: `#FFFFFF`
  - `accent`: `#6B75E8`
  - `text`: `#1A1D3A`
  - `muted`: `#5A5F7D`
- Типографика:
  - `display`: SF Pro Display Semibold
  - `body`: Inter Regular
- Визуальные маркеры:
  - высокая читаемость, низкая утомляемость
  - карточки с легкой тенью вместо неоновых контуров
  - контраст строится через типографику и иерархию отступов
- Лучшее применение:
  - дневная тема и onboarding-режим

## Direction C — Retro Deck

Идея: более «игровой» и запоминающийся стиль с теплым акцентом и карточной атмосферой.

- Палитра:
  - `bg`: `#1B1025`
  - `surface`: `#2A1A38`
  - `accent`: `#F2B441`
  - `secondary`: `#70D6FF`
  - `danger`: `#FF6B6B`
- Типографика:
  - `display`: SF Pro Display Bold
  - `body`: SF Pro Text Regular
- Визуальные маркеры:
  - теплые акцентные CTA-кнопки
  - чуть более плотные карточки и бейджи
  - выразительные статусы (онлайн, шулер, результат)
- Лучшее применение:
  - seasonal event/турнирный режим

## Единые требования для всех направлений

- Нижняя навигация: 3 фиксированные кнопки `Играть / Создать игру / Профиль`.
- Минимальный размер основного текста: `16px`.
- Единый формат валюты: `$123.45`.
- Состояния компонентов: `default / active / pressed / disabled`.
- Мобильная адаптация: ширины `320–430px`, safe-area Telegram.
