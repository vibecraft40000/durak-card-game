# Subscription gate (обязательная подписка на канал)

Чтобы открыть доступ к Mini App только после подписки на Telegram-канал, используются проверка на боте и на бэкенде.

## Требования

1. **Бот должен быть администратором канала.** Иначе метод `getChatMember` не сможет проверить участников.
2. **ID канала** — числовой (например `-1001234567890`). Для канала с invite-ссылкой вида `t.me/+P_rbSS0y5N9jM2Qy` публичного @username нет; ID можно узнать, добавив бота в канал как админа и посмотрев обновления (например, через getUpdates при каком-либо событии в канале) или через сторонние инструменты.

## Конфигурация

### Бот (`bot/.env`)

- `REQUIRED_CHANNEL_ID` — ID канала (число как строка, например `-1001234567890`). Если пусто — проверка подписки в боте выключена.
- `CHANNEL_INVITE_LINK` — ссылка для кнопки «Подписаться» (например `https://t.me/+P_rbSS0y5N9jM2Qy`).

### Бэкенд (`backend/.env`)

- `REQUIRED_CHANNEL_ID` — тот же ID канала. Если задан, при выдаче JWT (`POST /auth/telegram`) проверяется членство пользователя в канале; при отсутствии подписки возвращается 403 `subscription_required`.
- `SUBSCRIPTION_CHANNEL_LINK` — ссылка на канал для ответа API (по умолчанию используется ссылка из задания).

## Поведение

1. **В боте (`/start`):** если задан `REQUIRED_CHANNEL_ID`, бот проверяет подписку через `getChatMember`. Если пользователь не подписан — показывается сообщение с кнопками «Подписаться» и «Проверить подписку»; кнопка «Играть» не показывается. После подписки пользователь нажимает «Проверить подписку»; при успешной проверке показывается приветствие и кнопка «Играть».
2. **При прямом открытии Mini App (без бота):** фронт отправляет `initData` на `POST /auth/telegram`. Бэкенд при заданном `REQUIRED_CHANNEL_ID` проверяет подписку; при отсутствии подписки возвращает 403 с кодом `subscription_required`. Фронт показывает экран «Требуется подписка» с ссылкой на канал и кнопкой «Проверить подписку» (повторный запрос auth после подписки).

Таким образом обход через прямое открытие Mini App без подписки закрыт на стороне бэкенда.
